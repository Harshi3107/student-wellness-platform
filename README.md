# Student Wellness Platform (Frontend)

A frontend web application focused on improving student well-being by providing access to support resources, role-based dashboards, and structured interaction between students, counselors, and administrators.

This project was built as part of a student-driven initiative to explore user-centric design, accessibility, and real-world problem solving using modern frontend technologies.
